# 榕慧小程序前端项目

## 项目结构

```
huiliaoMiniProgram/
├── miniprogram/
│   ├── pages/
│   │   ├── home/           # 首页
│   │   ├── tongue-upload/  # 舌诊上传页面
│   │   ├── chen-agent/     # 陈主任智能体
│   │   └── ...
│   ├── utils/
│   │   └── util.ts         # 工具函数
│   ├── assets/
│   │   └── ...            # 图片资源
│   ├── app.json            # 小程序配置
│   ├── app.ts              # 小程序入口
│   └── ...
├── typings/
└── ...
```

## 主要功能

1. **舌诊上传**：
   - 上传舌苔视频进行分析
   - 支持选填舌上、舌下、面部照片
   - 实时显示分析结果

2. **智能问答**：
   - 陈主任智能体
   - 小慧智能体

3. **其他功能**：
   - 首页
   - 工作台
   - 专家页面
   - 报告上传

## 开发环境

- 微信开发者工具
- TypeScript
- 微信小程序框架

## 本地开发

1. **打开项目**：
   - 使用微信开发者工具打开 `huiliaoMiniProgram` 目录

2. **配置域名**：
   - 在开发者工具中，点击「详情」→「本地设置」
   - 勾选「不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书」

3. **启动后端服务**：
   - 运行后端代理服务（详见后端 README）

4. **编译运行**：
   - 点击开发者工具中的「编译」按钮
   - 在模拟器中测试功能

## API 调用

- **舌诊分析**：`http://127.0.0.1:8020/api/tongue-analysis`
- **智能问答**：`http://127.0.0.1:8010/api/chat`

## 后端服务

- **舌诊上传服务**：`http://127.0.0.1:8020`
- **智能体服务**：`http://127.0.0.1:8010`

## 注意事项

1. **视频要求**：
   - 大小不超过 5MB
   - 时长 10-20 秒
   - 帧率大于 2fps
   - 画面中需要有人脸和舌头

2. **开发模式**：
   - 本地开发时使用 HTTP 地址
   - 生产环境需要配置 HTTPS 域名

3. **权限设置**：
   - 需要相机、相册权限
   - 需要网络访问权限

## 部署说明

1. **代码上传**：
   - 在微信开发者工具中点击「上传」
   - 填写版本号和更新说明

2. **域名配置**：
   - 在微信公众平台「开发 → 开发设置 → 服务器域名」中添加合法域名

3. **发布审核**：
   - 提交审核
   - 审核通过后发布
